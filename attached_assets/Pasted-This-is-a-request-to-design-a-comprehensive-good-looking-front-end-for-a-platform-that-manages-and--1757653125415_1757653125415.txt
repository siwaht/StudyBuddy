This is a request to design a comprehensive, good-looking front-end for a platform that manages and analyzes AI voice agents built using ElevenLabs and connected via LiveKit. The platform needs to provide detailed analytics, call data access, and administrative controls.

Below are high-fidelity visual mockups (using SVG) illustrating the design for the key screens: the Dashboard, the Call Detail View (with in-depth analysis), and the Admin User Management screen.

### Design Philosophy

The design utilizes a modern, professional aesthetic with a dark sidebar and a light content area for optimal data visualization and readability. The focus is on usability, clarity, and providing deep insights relevant to voice agent performance.
